
🚀 Projeto desenvolvido como desafio final da formação Full Stack da Rocketseat e expandido com funcionalidades adicionais além dos requisitos originais.

🎓 Contexto do Projeto
Este projeto foi desenvolvido como parte do desafio final de certificação da Rocketseat, uma das maiores plataformas de educação em tecnologia do Brasil.
O objetivo do desafio foi aplicar conhecimentos de desenvolvimento Full Stack na construção de uma aplicação completa de Help Desk, contemplando regras de negócio, autenticação, gerenciamento de usuários, controle de acesso por perfis e integração entre frontend e backend.
Além dos requisitos propostos no desafio, foram implementadas diversas melhorias extras, incluindo:

Upload e gerenciamento de avatar
Alteração de senha
Controle de disponibilidade de técnicos
Responsividade completa para dispositivos móveis
Sidebar adaptativa
Feedback visual para autenticação e validações
Deploy completo em produção
Melhorias de experiência do usuário (UX)
