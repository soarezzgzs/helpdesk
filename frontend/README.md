# 🎨 HelpDesk Frontend

Frontend da aplicação HelpDesk desenvolvido com React, TypeScript e Vite.

Projeto desenvolvido como desafio final da formação Full Stack da Rocketseat.

---

## 🚀 Tecnologias

- React
- TypeScript
- Vite
- React Router DOM
- Axios
- React Hook Form
- Zod
- TailwindCSS
- Lucide React

---

## 📱 Funcionalidades

### Cliente

- Login
- Cadastro
- Criação de chamados
- Consulta de chamados
- Visualização de detalhes
- Edição de perfil
- Upload de avatar
- Alteração de senha

### Técnico

- Visualização de chamados atribuídos
- Atualização de status
- Inclusão de serviços adicionais
- Gerenciamento de disponibilidade
- Atualização de perfil

### Administrador

- Gestão de chamados
- Gestão de clientes
- Gestão de técnicos
- Gestão de serviços

---

## 📱 Responsividade

A aplicação foi desenvolvida utilizando abordagem Mobile First.

Suporte para:

- Smartphones
- Tablets
- Notebooks
- Monitores Full HD
- Monitores Ultra Wide

## ⚙️ Variáveis de Ambiente

Crie o arquivo:

```env
.env.production
```

```env
VITE_API_URL=https://helpdesk-o88p.onrender.com
```

---

## 🚀 Executando Localmente

Instalar dependências:

```bash
npm install
```

Executar aplicação:

```bash
npm run dev
```

Build de produção:

```bash
npm run build
```
